# Quiz de Empresas

Este proyecto es un ejercicio de programación en Python donde se procesan datos de empresas a partir de un archivo de texto. El programa lee las empresas registradas, filtra las que pertenecen a un sector específico (por ejemplo, TECNOLOGÍA) y calcula el promedio de empleados de las empresas registradas.

## Requisitos

- Python 3 instalado.
- Un archivo `empresas.txt` con datos en formato: `nombre, sector, empleados`.

## Uso

1. Coloca el archivo `empresas.txt` en el mismo directorio que el script Python.
2. Ejecuta el script con Python:
   ```bash
   python nombre_del_script.py
   ```
3. El programa debe mostrar las empresas filtradas y el promedio de empleados.

## Formato de datos

Cada línea de `empresas.txt` debe tener el formato:

```text
nombre, sector, empleados
```

Ejemplo:

```text
Tech Solutions, TECNOLOGÍA, 120
Alimentos S.A., ALIMENTOS, 45
```

## Objetivo

Practicar el manejo de archivos, procesamiento de texto y cálculos básicos en Python.
